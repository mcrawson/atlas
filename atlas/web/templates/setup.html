Environment Variables

All secrets are loaded from environment variables. You can set them in:

  • .env file in your ATLAS directory
  • Your shell profile (~/.bashrc, ~/.zshrc)
  • System environment variables

Testing Setup

After configuring, restart ATLAS and return to this page to verify your setup.

source .env && python -m atlas.web
